## 功能特性

### 一、个股量化分析（25+技术指标）

| 类别 | 指标 |
|------|------|
| **趋势** | MA5/10/20/60/120、均线多头/空头排列、金叉/死叉判断 |
| **动量** | RSI(14)、RSI(28)、MACD(DIF/DEA/柱)、KDJ(K/D/J) |
| **波动** | 布林带、ATR(14)、20日/60日波动率 |
| **量价** | 成交量、5日均量、20日均量、量比 |
| **支撑压力** | 近20/60/120日高低点、关键支撑/压力位 |
| **综合** | 综合评分(6分制)、年初至今涨跌、日内振幅 |

### 二、智能选股器（5条件评分）

从A股股票池（1481只）中按以下条件筛选超跌反弹候选股：

| 条件 | 说明 |
|------|------|
| RSI反弹区 | RSI(14) 处于 30-45 区间（超卖区域） |
| MACD蓄力 | MACD绿柱连续收缩 ≥ 3天，且 DIF<0（空方动能衰竭） |
| 量能放大 | 近3日均量 > 20日均量 × 1.3 |
| KDJ拐头 | KDJ的J值从 < 20 位置转头向上 |
| MA20支撑 | 收盘价 > MA20（趋势不破位） |

每满足1项 +1分，**≥2分** 即纳入候选池。

## 使用方法

### 个股量化分析

```bash
python3 scripts/a_stock_analysis.py <股票名称或代码>
```

**示例：**

```bash
# 按名称分析
python3 scripts/a_stock_analysis.py 贵州茅台
python3 scripts/a_stock_analysis.py 中国平安
python3 scripts/a_stock_analysis.py 比亚迪

# 按代码分析
python3 scripts/a_stock_analysis.py 600519
python3 scripts/a_stock_analysis.py 000333
```

### 智能选股器

```bash
# 5条件评分筛选（腾讯财经K线 + 新浪批量报价）
python3 scripts/stock_selector.py

# 可选：输出JSON结果
python3 scripts/stock_selector.py results.json
```

## 技术架构

### 数据源

| 接口 | 用途 |
|------|------|
| 腾讯财经 K线API | `web.ifzq.gtimg.cn` — 前复权日K历史数据（推荐，稳定无456限流） |
| 新浪财经报价 | `hq.sinajs.cn` — 当日实时价格/成交量 |
| 东方财富搜索API | `searchapi.eastmoney.com` — 股票代码搜索 |

### 文件结构

```
a-stock-analysis/
├── SKILL.md                      # OpenClaw技能说明
├── README.md                     # 本文档
├── package.json                  # 包配置
├── _meta.json                    # 元数据
└── scripts/
    ├── a_stock_analysis.py      # 🏠 主分析脚本（25+指标，腾讯K线）
    ├── analyzer.py               # 技术指标计算模块
    ├── stock_selector.py         # 选股器脚本（5条件评分，腾讯K线）
    └── fetch_hs300.py            # A股股票池列表
```

### 核心指标计算

- **MA**：简单移动平均
- **EMA**：指数移动平均（用于MACD）
- **RSI**：`100 - 100/(1 + AG/AL)`，AG/AL为周期内平均涨跌
- **MACD**：`EMA(12) - EMA(26)`，信号线为MACD的EMA(9)
- **KDJ**：RSV = (C - LL)/(HH - LL) × 100，K/D为RSV的3日平滑
- **布林带**：`MID = MA(20)`，`STD = sqrt(Var)`，`UPPER = MID + 2×STD`
- **ATR**：`TR = max(H-L, |H-PC|, |L-PC|)` 的14日平均
