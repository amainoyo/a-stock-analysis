# A股量化分析工具 (a-stock-analysis)

## 功能
1. **个股分析** — 25+ 项技术指标深度分析
2. **批量选股** — 从A股股票池（1481只）中按5个严格条件筛选

## 触发方式
```
"分析比亚迪"
"量化分析中国平安"
"帮我选股"
"帮我筛选超跌反弹的股票"
"帮我看看贵州茅台现在能买吗"
```

## 功能一：个股量化分析

### 25+ 分析指标
| 类别 | 指标 |
|------|------|
| 趋势 | MA5/10/20/60/120、均线多头/空头排列、金叉/死叉 |
| 动量 | RSI(14)、RSI(28)、MACD(DIF/DEA/柱)、KDJ(K/D/J) |
| 波动 | 布林带、ATR(14)、20日/60日波动率 |
| 量价 | 成交量、5日均量、20日均量、量比 |
| 支撑压力 | 近20/60/120日高低点、关键支撑/压力位 |

## 功能二：智能选股器（5项评分·严格版）

| 条件 | 说明 |
|------|------|
| RSI反弹区 | RSI(14) 处于 25-40 区间（精准超卖） |
| MACD蓄力 | MACD绿柱连续收缩 ≥ 4天，且 DIF<0（底部充分蓄力） |
| 量能放大 | 近3日均量 > 20日均量 × 1.5（真放量过滤） |
| KDJ拐头 | KDJ的J值从 < 15 位置转头向上（深度超卖反弹） |
| MA20支撑 | 收盘价 > MA20（趋势不破位） |

每满足1项 +1分，**≥3分**即纳入候选池（严格标准，噪音少）

### 选股输出示例
```
共扫描 1474 只，符合条件 39 只（评分≥3）
#1 隆基绿能(601012)  3/5  RSI14=33.8 + MACD绿柱收缩5天 + KDJ低位拐头(J=-1.5)
#2 中国银河(601881)  3/5  RSI14=28.5 + MACD绿柱收缩5天 + KDJ低位拐头(J=14.1)
#3 远光软件(002063)  3/5  RSI14=39.8 + MACD绿柱收缩4天 + KDJ低位拐头(J=10.0)
```

## 技术实现
- **报价数据源**：新浪财经批量报价 `hq.sinajs.cn`
- **K线数据源**：腾讯财经 `web.ifzq.gtimg.cn` 前复权日K（推荐，稳定无456限流）
- **分析**：纯 Python 3 标准库，无需 pip install
- **并发**：支持20并发获取，1481只股票约10秒完成
