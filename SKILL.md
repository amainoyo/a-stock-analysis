# A股量化分析工具 (a-stock-analysis)

## 功能
1. **个股分析** — 25+ 项技术指标深度分析
2. **批量选股** — 从沪深300成分股中按5个条件筛选

## 触发方式
```
"分析比亚迪"
"量化分析中国平安"
"从沪深300里选股"
"帮我筛选超跌反弹的股票"
"帮我看看贵州茅台现在能买吗"
```

## 功能一：个股量化分析

### 25+ 分析指标
| 类别 | 指标 |
|------|------|
| 趋势 | MA5/10/20/60/120、均线多头/空头排列、金叉/死叉 |
| 动量 | RSI(14)、RSI(28)、MACD(DIF/DEA/柱)、KDJ(K/D/J) |
| 波动 | 布林带、ATR(14)、20日/60日波动率 |
| 量价 | 成交量、5日均量、20日均量、量比 |
| 支撑压力 | 近20/60/120日高低点、关键支撑/压力位 |

## 功能二：智能选股器（5项评分）

| 条件 | 说明 |
|------|------|
| RSI反弹区 | RSI(14) 处于 30-45 区间（超卖区域） |
| MACD蓄力 | MACD绿柱连续收缩 ≥ 3天（空方动能衰竭） |
| 量能放大 | 近3日均量 > 20日均量 × 1.3 |
| KDJ拐头 | KDJ的J值从 < 20 位置转头向上 |
| MA20支撑 | 收盘价 > MA20（趋势不破位） |

每满足1项 +1分，**≥2分**即纳入候选池

### 选股输出示例
```
共扫描 11 只，符合条件 2 只
#1 中信证券(600030) 评分:2/5 [RSI反弹区 + KDJ低位拐头]
#2 华泰证券(601688) 评分:2/5 [RSI反弹区 + KDJ低位拐头]
```

## 技术实现
- **数据源**：东方财富 `push2his.eastmoney.com` K线API
- **网络**：Agent 的 `web_fetch` 工具获取数据
- **分析**：纯 Python 3 标准库，无需 pip install
